Ballgame
========

This is a two-dimensional physics library and a collection of applets demonstrating ball-shaped motion and collision physics. The [LÃ–VE](https://love2d.org/) 2D game engine is required to run these applets.

Building
--------

Use GNU Make to construct the ".love" files.

    make

Clean up after you're done.

    make clean

Lutro
-----

The applets also run on the [LibRetro](https://www.libretro.com/) core [Lutro](https://lutro.libretro.com/), but there is no controller interface yet.
